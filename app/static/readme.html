MissZhang

一个简洁、移动端友好的Python Web服务

项目特性

  • 响应式H5前端界面（移动端优先）
  • 联系表单后端API（/api/contact）
  • SQLite数据存储
  • 健康检查接口（/health）
  • 生产环境就绪的Gunicorn配置
  • 支持域名访问

技术栈

Python 3.9+ Flask SQLite Gunicorn HTML5 CSS3 JavaScript

部署信息

本应用已部署在云服务器上，使用以下配置：

  • 端口：80（标准HTTP端口）
  • 进程管理：Gunicorn守护进程
  • 日志：自动记录访问和错误日志
  • 数据库：SQLite文件存储

API接口

  • GET / - 首页
  • GET /health - 健康检查
  • POST /api/contact - 提交联系表单
